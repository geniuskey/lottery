🎱 게임 방법

🎯 게임 목표

참여자 수만큼 공이 핀볼 머신 위에서 떨어집니다. 핀과 범퍼에 튕기며 내려오는 공을 하단의 자동 플리퍼가 쳐 올려주지만, 가운데 배수구에 빠지면 탈락! 마지막까지 살아남는 공이 승리합니다.

👥 참여자 설정

쉼표(,) 또는 줄바꿈으로 구분하여 참여자 이름을 입력하세요. 이름*3 형식으로 같은 이름을 여러 개 만들 수 있습니다. 최소 2명 이상 필요합니다.

🏆 최종 생존자 수

슬라이더로 최종 생존자 수를 설정합니다. 해당 인원만 남으면 게임이 종료됩니다.

⚡ 게임 진행

시간이 지날수록 중력이 강해지고 플리퍼가 점점 게을러집니다. 30초가 넘으면 서든데스! 플리퍼가 멈춰 승부가 빠르게 결정됩니다.

🕹️ 플리퍼 조작

플리퍼는 기본적으로 자동으로 움직이지만, 직접 조작하면 사용자 입력이 우선합니다. 키보드 ←/Z는 왼쪽, →/X는 오른쪽 플리퍼를 작동시키며, 모바일에서는 게임 화면의 좌/우 절반을 누르면 됩니다. 누르고 있는 동안 플리퍼가 올라간 상태로 유지됩니다. 단, 서든데스 중에는 조작할 수 없습니다.

📋 결과 복사

게임 종료 후 하단의 복사 버튼으로 결과를 클립보드에 복사할 수 있습니다.

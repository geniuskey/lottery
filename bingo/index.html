빙고 노바 방법

  1. 참가자, 번호 범위와 필요한 빙고 줄을 설정합니다.
  2. 각 카드는 서로 다른 24개 번호와 무료 중앙 별을 받습니다.
  3. 볼 머신이 번호를 호출하면 모든 카드가 자동으로 체크됩니다.
  4. 동시에 빙고가 나오면 매 게임 새롭게 정한 타이브레이커가 순위를 정합니다.
게임을 다시 시작할 때마다 카드와 번호 호출 순서가 새롭게 무작위로 정해집니다.
